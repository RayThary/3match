# Block Pang / 3Match Puzzle Project

Unity 기반 3매치 퍼즐 개인 프로젝트입니다.

## 프로젝트 개요
- 장르: 3매치 퍼즐
- 개발 엔진: Unity 2022.3.4f1
- 개발 기간: 약 1달 2주
- 목표: 블록을 3개 이상 매칭하여 목표 점수를 달성하는 퍼즐 게임 구현

## 주요 구현
- 3매치 판정 로직
- 블록 선택 및 스왑 기능
- 매칭된 블록 제거 및 빈 공간 리필 흐름
- 4개 매칭 시 특수 블록 생성
- 스테이지별 블록 수 및 보드 크기 변경

## 기술 포인트
- 보드 배열을 기준으로 블록 상태를 관리
- 각 블록이 가로/세로 방향으로 같은 타입인지 체크하여 매칭 여부 판단
- 매칭된 블록을 리스트로 수집한 뒤 배열에서 제거하고 오브젝트 삭제
- 빈 배열 위치를 기준으로 위쪽 블록을 내려오게 처리
- 더 이상 매칭이 없을 때 새 블록을 생성하는 흐름 구현
- 인접한 블록만 스왑 가능하도록 좌표 비교 처리
- 스왑 후 매칭이 없으면 원래 위치로 되돌리는 로직 구현
